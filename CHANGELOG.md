# Changelog

## 0.1.0 - 2026-03-08

- Initial public release
- Added shared Python core for workbook loading, layout math, and PDF generation
- Added CLI entrypoint for generating PDFs from `.xlsx`
- Added local web UI for upload, preview, and PDF export
- Added English/Korean full-page language toggle
- Added dynamic role-style controls with default `SPEAKER + STAFF` rows and expandable rows up to 6
- Added public-safe dummy sample workbook and sample PDF
- Added regression tests for loader, layout, CLI, rendering, and web UI
